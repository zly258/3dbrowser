# 3D Browser Example App

[Live Preview](https://zly258.github.io/3dbrowser/)

This is a standalone example application demonstrating how to use the **@zhangly1403/3dbrowser** library.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

## Deployment

This repository is configured to deploy automatically to **GitHub Pages** using GitHub Actions.
Check `.github/workflows/deploy.yml` for details.

## Dependencies

- **@zhangly1403/3dbrowser**: The core 3D viewer library.
- **Vite**: Build tool.
- **React**: UI framework.
